# 帳友 Account Buddy

帳友是一個以個人與多人記帳為核心的網頁專案，目標是讓使用者可以記錄日常收入、支出、帳戶、幣別，以及多人分帳時的金錢往來。

本專案目前作為前端練習、作品集專案，以及未來專題開發的基礎。

---

## 目前功能

- 首頁版面設計
- 側拉選單規劃
- 登入 / 註冊頁面規劃
- 單人記帳功能規劃
- 多人記帳功能規劃
- 金錢糾紛 / 欠款統計功能規劃
- 幣別、帳戶、分類自訂功能規劃

---

## 使用技術

- HTML
- CSS
- JavaScript
- Firebase Auth
- Firebase Firestore

---

## 專案目標

本專案希望建立一個簡單、直覺、可擴充的記帳網站，未來可以支援：

- 個人收支紀錄
- 多人分帳
- 自訂帳戶
- 自訂分類
- 自訂幣別
- 匯率換算
- 登入後個人化資料管理

---

## 專案狀態

目前仍在開發初期，主要進行前端頁面設計與功能架構規劃。
